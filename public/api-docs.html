API Documentation

راهنمای کامل استفاده از API استودیو هوش مصنوعی مد

احراز هویت

برای استفاده از API، ابتدا باید ثبت نام کنید و توکن دریافت کنید.

POST /api/auth/signup

Body:

{ "email": "user@example.com", "password": "your_password" }
POST /api/auth/signin

Response:

{ "success": true, "session": { "access_token": "eyJhbGc...", "user": { ... } } }
POST /api/upload

آپلود تصویر لباس

آپلود فایل تصویر برای پردازش

Headers:

Content-Type: multipart/form-data Authorization: Bearer {access_token}

Form Data:

پارامتر نوع الزامی توضیحات
garment File ✓ فایل تصویر (JPG, PNG, WEBP, AVIF - حداکثر 10MB)

Response:

{ "success": true, "filePath": "https://...", "fileName": "1234567890-shirt.jpg" }
GET /api/models

دریافت لیست مدل‌ها

لیست تمام مدل‌های موجود (زن، مرد، دختر، پسر)

Response:

[ { "id": "woman-1", "name": "مدل ۱", "category": "woman", "categoryName": "زن", "description": "زن 35 ساله", "image": "https://..." }, ... ]
GET /api/backgrounds

دریافت لیست پس‌زمینه‌ها

لیست تمام پس‌زمینه‌های موجود

Response:

[ { "id": "studio-white", "name": "استودیو سفید", "description": "Professional white studio background", "image": "https://..." }, ... ]
POST /api/generate

تولید عکس با هوش مصنوعی

تولید عکس استودیویی از لباس روی مدل

Headers:

Content-Type: application/json Authorization: Bearer {access_token}

Body Parameters:

پارامتر نوع الزامی توضیحات
garmentPaths Array ✓ آرایه URL های تصاویر لباس
modelId String ✓ شناسه مدل (woman-1, man-1, ...)
backgroundId String ✓ شناسه پس‌زمینه
poseId String حالت بدن (پیش‌فرض: standing-front)
styleId String استایل عکس (پیش‌فرض: professional)
aspectRatioId String نسبت تصویر (پیش‌فرض: 1:1)

Request Example:

{ "garmentPaths": ["https://..."], "modelId": "woman-1", "backgroundId": "studio-white", "poseId": "standing-front", "styleId": "professional", "aspectRatioId": "1:1" }

Response:

{ "success": true, "imagePath": "https://...", "model": "مدل ۱", "background": "استودیو سفید", "description": "...", "message": "تصویر با موفقیت تولید شد!" }
POST /api/generate-caption

تولید کپشن اینستاگرام

تولید کپشن جذاب فارسی برای تصاویر

Body:

{ "imageUrl": "https://...", "imageId": "uuid" }

Response:

{ "success": true, "caption": "کپشن تولید شده...", "imageId": "uuid" }
GET /api/generations

دریافت تاریخچه تولیدها

لیست 100 تصویر آخر تولید شده توسط کاربر

Response:

{ "success": true, "generations": [ { "id": "uuid", "garment_path": "https://...", "generated_image_url": "https://...", "created_at": "2024-01-01T00:00:00.000Z", ... }, ... ] }
